# DIGI-1665
A web app to digitalize a mechanic workshop. Clients can book appointments online instead of calling.
Built with Python/Flask (API) + MariaDB (database) + HTML/CSS/JS (frontend), all running inside Docker containers.
bashdocker compose up --build
Web → http://localhost:8080
API → http://localhost:5000
